The factory theme demonstrates factory products and collect quotes from customers. It consists of products, news, pages, inquiries and SEO optimization settings.

The admin can set favicon, logo, title etc in all its theme settings.

The admin can publish news, products, collect customer inquiries and quotes in one website.

The theme takes advantage of codestar framework to build its backend input options.
